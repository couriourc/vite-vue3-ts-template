# Vue 3 + TypeScript + Vite

这个模板应该可以帮助你开始在 Vite 中使用 Vue 3 和 TypeScript 进行开发。 该模板使用 Vue 3 `<script setup>` SFC，查看更多信息：[script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) 。
